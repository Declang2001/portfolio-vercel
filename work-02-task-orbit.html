case study

task orbit

ai powered task app that turns pasted content into a clean actionable task list.

role: design + frontend + ai integration tools: figma, html/css, javascript timeline: concept sprint output: prototype + visuals

the ask

create an app that turns scattered tabs, emails, and notes into one focused system.

my role

end to end build. design ux and motion, build the ui, and wire up task extraction.

output

a polished console that converts messy inputs into structured tasks.

task orbit reimagines how people capture and organize work. instead of manually sorting through tabs and messages, users paste raw content and the app extracts structured, actionable tasks automatically.

a minimal console interface that feels calm despite handling complex input. the extraction pipeline surfaces priorities and deadlines without requiring the user to think about structure.

designed the ux flow and visual language in figma, then built the frontend with vanilla html, css, and javascript. the task extraction logic was wired up to parse and categorize pasted content into clean task cards.
