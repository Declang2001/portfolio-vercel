case study

pacifico dance company

reframed how the brand communicates across motion, web direction, and presenter-facing touchpoints.

role: brand direction + motion + website implementation tools: premiere pro, after effects, blender, github + terminal timeline: ongoing collaboration deliverable: short-form edits + animated identity + website refresh

brief

pacifico needed a clearer way to present itself so first-time visitors and presenters could read the brand faster.

build

shaped the communication layer through motion, content direction, and visual system work so the brand felt more organized and intentional across touchpoints.

result

a stronger presentation system the brand uses to communicate more clearly across platforms and inquiry-facing moments.

pacifico needed a clearer, more cohesive way to present itself. the communication came together through short-form video, motion graphics, and branded visual direction — giving the company a more consistent read across platforms.

the strongest part of this project was building a reusable communication layer, not just one-off assets. the motion work, content system, and visual direction all helped the brand present itself more clearly and consistently.

started by reviewing the existing brand presence and identifying where the communication felt fragmented or unclear. then built motion, edit, and visual-direction assets that made the brand easier to read, more cohesive, and more usable across touchpoints.
